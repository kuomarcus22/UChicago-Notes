## LiveTeXing Setup

These notes are typeset in real time during lectures and minimally cleaned up afterwards (usually as I go through them when studying for midterms/finals).

I use Neovim with VimTeX, Skim, and UltiSnips. Snippets are what allow me to LiveTeX.

The snippets started from Gilles Castel's [guide](https://castel.dev/post/lecture-notes-1/) on LiveTeXing, which seems to have helped many people. 
The full snippet file is at `snippets/tex.snippets`. It's a bit messy and idiosyncratic; I would recommend starting with Castel's snippets and building your own as you see fit.
